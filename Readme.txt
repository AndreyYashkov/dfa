Требования к коду
Всё сделать в виде АПИ (в DRF)
CRUD к картинкам
Сделать роут для удаления всех картинок из базы для администратора
Любая авторизация, регистрация
Роут на получение текущего юзера

______________________________________________________


Старт:
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver


Регистрируем юзера:
http://127.0.0.1:8000/auth/users/
Поля (Email address, Username,Password)


добавляем фото, видим их список:
http://127.0.0.1:8000/api/v1/Photos/

Есть возможность Detatil просмотра через
http://127.0.0.1:8000/api/v1/Photos/ __Id записи для изменения конкретного фото__ с пермишинами isownerorreadonly




Только Админ может выполнить этот экшен для удаления всех фоток из базы
http://127.0.0.1:8000/api/v1/Photos/delete_all/

Сыылка на текущего пользователя:
http://127.0.0.1:8000/auth/users/me/
